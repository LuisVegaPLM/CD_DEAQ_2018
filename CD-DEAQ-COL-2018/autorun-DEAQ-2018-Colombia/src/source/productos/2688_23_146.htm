DESCRIPCIÓN: COLJAP® ZINC es un fertilizante mezclado granulado de alta concentración de zinc, formulado a base de sulfatos para aplicación directa al suelo solo o en mezcla con fertilizantes NPK, ideal para suministrar las necesidades de zinc de diferentes cultivos y/o corregir deficiencias.
