DESCRIPCIÓN: POLIQUEL® MULTI es un fertilizante foliar quelatado de alta solubilidad, formulado con un complejo de varios agentes quelatantes o secuestrantes a base de EDTA y lignosulfonatos que acomplejan el magnesio, hierro, manganeso, y zinc. Su aplicación permite prevenir las deficiencias nutricionales de los elementos menores, y balancea la nutrición general de las plantas, para obtener mejores rendimientos y calidad de cosecha.
