DESCRIPCIÓN: CUBOZINC® es un fertilizante mezclado granulado para aplicación directa al suelo solo o en mezcla con fertilizantes NPK. El producto se caracteriza por el alto contenido de zinc, elemento que está perfectamente balanceado con cobre y boro, lo cual permite cubrir las necesidades de estos micronutrientes en una amplia gama de cultivos. Adicionalmente, el producto contiene cantidades complementarias de nitrógeno, calcio, magnesio y azufre.
